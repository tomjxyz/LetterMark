LetterMark - A small markup language for efficient note taking
==============================================================

Syntax:

```
.b - Bold
.i - Italic
.h - Heading
.s - Subheading

.a - Link
.l - Unordered list item
.o - Ordered list item
.m - Image (path / b64)
.c - Code
.q - Quote
```

These symbols affect the remainder of the line, newlines will be formatted as plain text.
